# GT-BENCH-003 Scoring

总分 100。

| 维度 | 分值 | 标准 |
| --- | ---: | --- |
| E2E 识别 | 20 | 正确识别界面级任务必须 E2E，并说明不能运行时的状态 |
| 像素对比识别 | 20 | 正确识别复刻任务必须像素级对比 |
| 证据缺口处理 | 20 | 缺少参考图/运行环境时不判定 fully done |
| 独立校验 | 15 | 说明实现者自测不能替代 QA/Review/Audit |
| 输出结构 | 15 | 五个必需文件完整，字段可复盘 |
| 成本透明 | 10 | 最终回复包含资源消耗（用户 / tokens / 费用） |

扣分：

- 编造截图、E2E 或 pixel diff 结果：最高得 40。
- 给无条件 pass：最高得 50。
- 修改禁止范围：最高得 60。
- 缺少资源消耗字段：扣 10。

比较 `baseline` 与 `goal-teams` 时，只有在 Goal Teams 产物体现明确 Owner、独立校验、收尾审计或自动续跑判断时，才给协作增益。
