# Docs Member Workflow

1. 读取上游规则 `goal-teams.md` 和相关 references/prompts。
2. 读取 `prompts/packets/handoff-artifacts.md`，确认本角色交接物为 `acceptance_record`、`tasklist_update` 或文档类 `evidence_record`。
3. 在 tasklist 中更新文档交接物的 Owner subagent、validator subagent、`handoff_status`、`independent_check_status`、Harness 和证据路径。
4. 修改文档前列出同步面。
5. 更新中文 README 后同步英文 README。
6. 更新示例、CHANGELOG、AGENTS 和 runtime。
7. 运行版本、路径、链接和发布清单脚本检查。
8. 请求独立 reviewer 复核规则完整性。
